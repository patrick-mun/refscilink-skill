Example host website for RefSciLink

Population Genetics Learning Project

This demonstration page is intentionally designed as a small scientific website. It is used to test how RefSciLink installs a bibliography module, detects a visual identity, and adds a references page without breaking the original design.

Demo Theme detection target

Primary color, rounded cards, navigation style and button geometry should be detected by the RefSciLink skill.

Objectives

Why this example exists

Test installation

Provide a clean host website where the command /create_module_ref can install the bibliography module.

Test theme adaptation

Expose a clear visual identity so the AI can infer colors, typography, spacing and border radii.

Test Markdown parsing

Use a mixed-format bibliography file with DOI, PMID, URL, preprint and incomplete references.

Methods

Expected RefSciLink behaviour

The skill should analyse bibliographie.md, detect the references section, create data/reference_bibliographique/, generate JSON files and add a Références button to the navigation or as a safe fallback button.

  1. Analyse the host website.
  2. Detect theme variables from CSS.
  3. Extract references from Markdown.
  4. Generate the bibliography module.
  5. Preserve the original design.

Results

What should be visible after installation

After running the RefSciLink AI-assisted installation, this page should contain a new Références entry. The generated bibliography page should visually inherit this turquoise scientific theme.
