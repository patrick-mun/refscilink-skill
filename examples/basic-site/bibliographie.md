# Bibliographie de test — Basic Site Example

Ce fichier Markdown sert de source de test pour le skill RefSciLink.

Il contient du texte général, puis une section de références bibliographiques volontairement hétérogène. Le but est de vérifier que l'assistant peut repérer les références dans un document qui ne contient pas uniquement une bibliographie.

## Contexte

Ce mini-site illustre un projet pédagogique fictif de génétique des populations. Les références ci-dessous couvrent des sujets utiles pour tester un module bibliographique scientifique : variation humaine, génomes de référence, populations admixées, preprints, bases de données et références incomplètes.

## Notes internes

Cette section ne doit pas être interprétée comme une bibliographie.
